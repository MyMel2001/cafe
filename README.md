# Cafe

A simple, scriptable alternative to Amphetamine.

## What this is
* A Simplified alternative.
* Without GUI (CLI ONLY!)
* Great for dev and AI workloads.
* Warm and friendly
* MIT Licensed!

## Some notes

* PID of current caffeinate instance always stored at "~/.cafe-pid"
* Kills all other "caffeinate" instance as to not conflict our current one.
